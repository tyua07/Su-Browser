基础

时区​

当前浏览器对应的时区

注入说明​

注入属性名称类型示例
basic.timezonestringAsia/Shanghai

历史记录数量​

浏览器初始化的历史记录数量

tip

在有些场景下，需要模拟之前就打开了很多个标签页面或者是访问了很多个页面才到达此页面的。使用该属性可以模拟已经打开过很多页面。

注入说明​

注入属性名称类型示例
basic.init-history-lengthint10

注入 JS​

可以模拟在页面打开前执行自定义的 js 代码。

注入说明​

注入属性名称类型示例
basic.inject-jsstring无

允许的权限​

把浏览器所支持的权限默认设置成允许。

tip

例如我们需要设置当前页面的剪切板的内容就需要开启剪切板权限，不然浏览器默认会提示需要用户允许才能获取剪切板内容。

注意

在示例中， 数字 5 代表是浏览器通知权限，但是随着浏览器内核的更新，对应的权限的枚举可能不一样。

注入说明​

注入属性名称类型示例
basic.allow-permissionsarray[5]

拒绝的权限​

把浏览器所支持的权限默认设置成拒绝。

注意

在示例中， 数字 4 代表是浏览器定位权限，但是随着浏览器内核的更新，对应的权限的枚举可能不一样。

注入说明​

注入属性名称类型示例
basic.reject-permissionsarray[4]

剪切板​

修改当前页面剪切板内容

注意

需要把剪切板的权限设置为允许，否者会浏览器会弹出一个授权框让用户允许才能获取剪切板内容的值。

注入说明​

注入属性名称类型示例
clipboard.textstring"这是一个强制注入的剪切板内容"

注入字体​

浏览器可以通过渲染字体来判断是否支持改字体。

注入说明​

注入属性名称类型示例
font.list-jsonarray["微软雅黑"]

浏览器名称​

浏览器名称

注意

注意：非必要请不要设置或者使用该属性。

注入说明​

注入属性名称类型示例
version-info.product-namestringGoogle Chrome

浏览器版本号​

浏览器版本号

注意

注意：该属性需要特别注意，因为浏览器会有多个 API 接口返回不一样的浏览版本号。所以我们会有一篇文章来说明浏览器版本应该怎么正确设置。

注入说明​

注入属性名称类型示例
version-info.numberstring"100.0.4896.127"

alert 弹框​

是否禁用 alert 弹框，如果禁用后，js 代码里的 alert 弹框将不会显示，这样自动化测试就不会因为 alert 而停止。

注入说明​

注入属性名称类型选项示例
frame.disable-alertint1：true；0：false1

window.open​

是否禁用 window.open 弹框，如果禁用后，js 代码里的 window.open 打开页面将不会生效。

注入说明​

注入属性名称类型选项示例
frame.disable-window-openint1：true；0：false1

注入说明​

注入属性名称类型选项示例
frame.confirmint1：true；0：false1
